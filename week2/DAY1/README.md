# Week 2 - Day 1 SQL Analysis

## Overview
This folder contains the SQL query files and output results for Day 1 of the Week 2 training.
The focus is on learning and practicing:
- Common Table Expressions (CTEs)
- SQL subqueries

## Contents
- `sql_CTEqueries.sql` - SQL scripts demonstrating Common Table Expressions.
- `sql_CTEqueries_outputs/` - Output results or supporting files generated from the CTE queries.
- `sql_subqueries.sql` - SQL scripts demonstrating subqueries.
- `sql_subqueries_outputs/` - Output results or supporting files generated from the subquery scripts.

## How to use
1. Open the `.sql` files in a SQL editor or database client.
2. Execute the queries against your training database.
3. Review the outputs in the corresponding `_outputs` folders to verify results.

## Notes
- The `_outputs` folders are intended to store query result snapshots or export files.
- If you want to add more analysis, create new SQL files and document them here.
